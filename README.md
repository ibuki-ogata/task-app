# TODOアプリ
階層構造・色管理・ドラッグ＆ドロップ対応のタスク管理アプリ

# 概要
このアプリは、
階層構造のタスク管理を中心に、
色選択・チェック伝播・ドラッグ＆ドロップなど、
実用的な機能を備えた Web ベースのタスク管理ツールです。

Notion や Todoist のような階層タスクを、
自分で一から実装した学習・作品用プロジェクトです。
(AIに協力してもらい作成しました)

## 主な機能

# 階層タスク
無限階層のタスク構造
折りたたみ（開閉）
親タスクの下に子タスクを追加可能

# 色選択UI
パレット選択
RGB入力（0〜255）
HEX変換
リアルタイムプレビュー
現在の色を RGB欄に自動反映
親の色を子が継承

# チェックの伝播
親にチェック → 子孫すべて完了
子のチェックを外す → 先祖すべて未完了
兄弟タスクは影響なし
完了タスクはグレーアウト表示

# ドラッグ＆ドロップで並び替え
SortableJS を使用
親の変更は不可

# メモ機能
タスクごとにメモを保存
モーダルで編集、普段は非表示

# バックエンド連携
Flask + SQLite
REST API でタスク CRUD
再帰クエリで階層処理（子孫取得・先祖取得）

## 使用技術
# フロントエンド
HTML / CSS / JavaScript
SortableJS（ドラッグ＆ドロップ）
color-mix / CSS変数
モーダル UI

# バックエンド
Python (Flask)
SQLite
再帰クエリ（WITH RECURSIVE）

## セットアップ方法
git clone https://github.com/ibuki-ogata/task-app.git
cd task-app
pip install -r requirements.txt
python app.py

ブラウザで
http://localhost:5000  
にアクセス。
